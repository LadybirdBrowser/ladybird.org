### CSS features

- **`if()`**: The `if()` function can now be used in property values, with the `supports()` and `media()` conditions both supported. ([#8244](https://github.com/LadybirdBrowser/ladybird/pull/8244))
- **`inherit()`**: We implemented `inherit()`, which allows property values to use the inherited value of any property. ([#8409](https://github.com/LadybirdBrowser/ladybird/pull/8409))
- **`@supports env(variable-name)`**: The `env()` function now works in `@supports`, allowing pages to query if Ladybird supports a given environment variable. ([#8645](https://github.com/LadybirdBrowser/ladybird/pull/8645))
- **`@font-feature-values`**: We now honor `@font-feature-values`, applying its instructions when rendering fonts. ([#8050](https://github.com/LadybirdBrowser/ladybird/pull/8050))
- **Pseudo-element custom properties**: We now correctly allow custom properties defined for the `::first-line`, `::first-letter`, and `::placeholder` pseudo-elements. ([#8267](https://github.com/LadybirdBrowser/ladybird/pull/8267))
